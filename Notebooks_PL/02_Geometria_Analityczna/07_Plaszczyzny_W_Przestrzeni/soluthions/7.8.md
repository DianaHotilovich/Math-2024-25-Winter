Musimy znaleźć równanie płaszczyzny przechodzącej przez trzy punkty $A = (2,0,0)$, $B = (0,3,0)$, $C = (0,0,4)$

Mamy ogólne równanie płaszczyzny

$$
ax+by+cz+d=0
$$

Musimy rozwiązać trzy różne równania, abyśmy mogli je następnie zrównać i dowiedzieć się, ile będą równe a b i c

Dla punktu A = $a*2+b*0+c*0 => 2a=d$

Dla punktu B $a*0+b*3+c*0 => 3b=d$

Dla punktu C $a*0+b*0+c*4 => 4c=d$

$$2a=3b=4c=d | * 12$$
 - ogólna liczba, której możemy użyć do znalezienia wartości a, b i c

 $12/2 = 6 => a$

 $12/3 = 4 => b$

 $12/4 = 3 => c$

Wynik:
 $$
6x+4y+3z=12
 $$
